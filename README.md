### Project Title
Project Title- Explore US bikeshare data

### Description
In this project, Python is used to explore data related to bike share systems for three major cities in the United States—Chicago, New York City, and Washington.Interesting questions are answered by computing descriptive statistics. Softwares such as Python 3, Numpy and pandas are needed for the completion of project.

### Files used
chicago.csv
new_york_city.csv
washington.csv
